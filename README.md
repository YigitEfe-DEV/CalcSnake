# CalcSnake

CalcSnake is a premium calculator built with React and Vite. It combines a polished scientific calculator with a hidden Snake mode that turns the calculator display into the game screen after a secret unlock sequence.

## Features

- Full calculator with keyboard support
- Scientific operations: percent, square root, power, and parentheses
- Calculation history panel with clear history action
- Error handling for invalid expressions
- Hidden Snake mode activated by entering `1984` and pressing `=`
- Snake controls with arrow keys and WASD
- Score and high score tracking with local persistence
- Pause, restart, and game over states
- Smooth motion, responsive layout, and dark glassmorphism styling

## Installation

```bash
npm install
```

## Usage

```bash
npm run dev
```

Open the local development URL shown in the terminal.
